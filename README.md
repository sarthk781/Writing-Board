# Writing Board

Write. Draw. Share.

Writing Board is a free online writing and drawing tool built using HTML, CSS and JavaScript. It allows users to write, sketch, draw and share ideas directly from their browser without installing any software.

## Features

* Pencil Tool
* Eraser Tool
* Rainbow Pen
* Color Picker
* Adjustable Brush Size
* Undo and Redo
* Save Drawing as PNG
* Share Drawing
* Dark Mode
* Grid Paper Mode
* Notebook Lines Mode
* Touchscreen Support
* Mobile Friendly Design
* Responsive Layout

## Use Cases

* Handwriting Practice
* Drawing and Sketching
* Online Whiteboard
* Classroom Activities
* Teaching and Learning
* Note Taking
* Creative Brainstorming

## Technologies Used

* HTML5
* CSS3
* JavaScript
* HTML Canvas API

## Getting Started

1. Clone the repository
2. Open the project folder
3. Open `index.html` in your browser
